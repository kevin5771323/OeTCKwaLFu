# 项目概述 java29

本项目是一个基于B/S架构的仓库管理系统，覆盖了从客户到订单处理，再到生产、质检、库存管理以及出库的完整企业管理链路。

## 技术栈

### 后端
- Spring Boot
- Spring Security
- MyBatis Plus
- Redis
- WebSocket

### 前端
- Vue
- ElementUI
- ECharts
- Vuex
- WebSocket

## 运行环境
- JDK 1.8
- MYSQL 8.0

## 功能模块
- 客户管理
- 订单管理
- 生产管理
- 待检管理
- 入库管理
- 盘点管理
- 出库管理

## 系统角色
- 系统管理员
- 普通用户

## 目录结构
```
项目目录
├── src
│   ├── main
│   │   ├── java
│   │   └── resources
│   └── test
└── web
    ├── backend
    └── frontend
```

## 核心亮点
1. **安全可靠**：整合Spring Security，实现了用户认证和权限管理。
2. **高效缓存**：整合Redis，用于权限认证时缓存用户角色权限数据。
3. **工作效率提升**：采用MyBatis Plus，简化了开发流程，提高了工作效率。
4. **物流跟踪**：通过快递鸟第三方接口，支持物流跟踪功能（注：免费使用，支持中通、圆通、申通）。
